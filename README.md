# Where’s Waldo — Frontend

Frontend for a **Where’s Waldo–style game** built with React.

Players select a level, enter a username, and race against the clock to find
hidden characters in large images. Correct clicks are validated using image
coordinates, and final times are shown on a leaderboard.

## Tech Stack

- React
- React Router
- Tailwind CSS
- Fetch API

## Features

- Multiple levels
- Click-based character detection
- Timer with millisecond precision
- Leaderboard per level
- Replay support

## Setup

```bash
npm install
npm run dev
```
